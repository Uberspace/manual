Java
####

Introduction
============

Java is a general-purpose, object-oriented programming language. Interally, it
uses a virtual machine, making "compiled" java programs very independent of the
actual hardware executing them.

We provide both the JRE / Java Runtime Environment (``java``) and the JDK / Java
Development Kit (``javac``).

Versions
========

We only provide the version available in EPEL 7 - ``java-latest-openjdk``.
